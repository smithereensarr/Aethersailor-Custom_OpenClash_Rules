📖 使用说明
  1. 在上方输入框粘贴你机场原始订阅链接（不用做任何编码处理）。
  2. 点击「生成」，下方会出现一条新的订阅链接。
  3. 点「复制」，把它添加到你的 Clash 客户端的订阅 / 配置里。
  4. 客户端拉取时会自动经本服务用 Blue 模板转换，生成带防 DNS 泄漏分流规则的配置。
